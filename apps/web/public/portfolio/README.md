# Portfolio images

The 10 `.jpeg` files in this directory are placeholder solid-color JPEGs
sized to match the eventual real photos. Replace each file in place with
the matching real photo (same filename, same orientation) and the site
will pick them up with no code changes.

| Filename | Subject |
| --- | --- |
| `marble-look-foyer-tile.jpeg` | Marble-look foyer and entry tile installation. |
| `marble-look-entry-tile-angle.jpeg` | Marble-look entry tile from alternate angle. |
| `open-plan-wood-look-tile-floor.jpeg` | Open-plan kitchen and living area with wood-look tile floor. |
| `patterned-kitchen-backsplash-wide.jpeg` | White patterned kitchen backsplash, wide view. |
| `patterned-kitchen-backsplash-detail.jpeg` | Kitchen backsplash detail around range and counters. |
| `fireplace-marble-tile-surround.jpeg` | Marble-look fireplace surround and hearth tile. |
| `arched-stone-shower-tile.jpeg` | Arched shower with dark stone-look wall tile and pebble floor. |
| `arched-shower-niche-bench-detail.jpeg` | Shower niche, bench, pebble floor, and stone-look wall tile detail. |
| `gray-shower-tile-bench-niche.jpeg` | Gray shower tile installation with bench and wall niche. |
| `diagonal-entry-tile-before.jpeg` | Entry tile project before / alternate existing condition. |

Image dimensions used in `next/image` are declared in
`apps/web/src/content/portfolio.ts`. If you swap to a different aspect
ratio, update the `width`/`height` there.
